배열 (Array)

        - 변수를 묶음으로 다루는 것(자료구조의 기초)
        (JS 변수의 특징 : 대입되는 값에 따라 자료형이 변함)

        JS 배열 : 자료형 상관 없이 여러 값을 한번에 다룸

        JS 배열 특징
        1. 자료형 제한 X (보통 한가지만 저장)
        2. 길이   제한 X (동적으로 크기 조정)

        [JS 배열 선언 방법]
        1. const arr = new Array(); //0칸 짜리 배열(비어있는 배열) 생성
        2. const arr = new Array(3); //3칸짜리 배열 생성, 각 칸의 값은 비어있음
        3. const arr = [값1, 값2, 값3]; // 3칸짜리 배열 생성, 각 칸이 값으로 채워져있음
        4. const arr = [] // 0칸 짜리 배열 생성
