이벤트(Event)

        이벤트 : 동작, 행위, (행사)
        -> 브라우저에서 할 수 있는 동작

        ex) click, keydown, keypress, keyup, mouseover, mouseon,mouseout, mouseleave,
        drag, change, submit,...

        HTML 이벤트 종류 참고
        * 이벤트 리스너(Event Listener)
        -> 브라우저에서 이벤트가 감지될 때 까지 대기
        감지되면 연결된 기능 수행

        - 이벤트 리스너 'on이벤트명' 으로 작성
        ex) onclick, onchange, onsubmit, onkeyup

        * 이벤트 핸들러(Event Handler)
        -> 이벤트 리스너에 연결된 기능(함수)로
        이벤트가 감지되었을 때 수행할 내용을 작성한 함수(function)

        < button onclick="버튼클릭기능()">클릭</button>
        
        onclick       : 이벤트 리스너
        버튼클릭기능(): 이벤트 핸들러

인라인 모델

        HTML요소에 이벤트 리스너="이벤트핸들러"형식을 작성하는 방법

        -하나의 요소에 이벤트 핸들러 중복 작성 X
            (이전에 작성된 이벤트 핸들러를 새로운 이벤트 핸들러가 덮어쓰기)

        인라인 이벤트 모델 작성시 this -> 이벤트가 발생한 현재 요소
